# Daegu Golden Time Governance

대구광역시 행정동별 응급의료 접근성과 보호가 필요한 인구를 함께 분석해, 응급환경 취약지를 지도와 데이터로 확인하는 프로젝트입니다. 일반 의료환경 전체가 아니라 응급실, 중증질환 수용 가능성, 야간·휴일 소아진료 공백에 초점을 둡니다.

## 핵심 관점

| 구분 | 내용 |
| --- | --- |
| 분석 지역 | 대구광역시 |
| 분석 단위 | 행정동, 응급기관, 정책 검토 후보지 |
| 시민 화면 | 현재 위치 기준 응급기관, 병상, 차량 이동시간, 중증질환 수용 가능 정보 |
| 정책 화면 | 취약 행정동, 실제 도로 이동시간, 새 거점 후보 비교 |
| 소아 영역 | 일반 소아과가 아니라 야간·휴일 소아진료 자원인 달빛어린이병원 기준 |
| 데이터 성격 | 공공데이터, 저장된 분석 자료, 길찾기 결과를 결합한 공공데이터 기반 시제품 |

## 개선 계기

초기 화면은 가까운 응급기관과 병상 정보를 중심으로 구성되어 있었습니다. 가족 피드백 과정에서 “응급환자용이라면 심근경색, 외상, 추락, 절단처럼 상황별로 갈 수 있는 병원이 달라야 한다”는 지적을 받았습니다.

이 피드백을 반영해 단순한 병원 목록이 아니라, 실제 응급 상황에서 확인해야 할 질문으로 화면을 바꾸었습니다. 현재 화면은 가까운 병원을 보여주는 동시에 중증질환 수용 가능 여부, 야간·휴일 소아진료 여부, 출발 전 전화 확인 필요성을 함께 보여줍니다.

## 주요 기능

- 현재 위치 기준 응급기관 목록과 지도 표시
- 실시간 응급실 병상 정보와 병상 상태 배지
- 차량 이동시간 보조 조회
- 중앙응급의료센터 공식 중증질환 수용 가능 정보 표시
- 심근경색, 뇌출혈, 외상/복부응급, 절단, 응급내시경, 중증화상, 야간·휴일 소아진료 필터
- 달빛어린이병원을 일반 소아과와 분리해 응급실 부담 완화 자원으로 표시
- 행정동 중심점에서 응급기관까지의 도로 이동시간 기반 접근성 분석
- 새 거점 후보별 평균 이동 부담과 15분/30분 안 도착 비율 표시

## 중증·응급 상황 필터

시민 화면은 단순히 가까운 병원을 보여주는 데서 멈추지 않고, 공식 데이터 기준으로 특정 중증질환 또는 처치 수용 가능 여부를 함께 보여줍니다.

| 화면 선택지 | 기준 |
| --- | --- |
| 심근경색 | 재관류중재술 심근경색 |
| 뇌출혈 | 뇌출혈 수술 |
| 외상/복부 | 대동맥 응급, 복부응급수술 관련 항목 |
| 절단 | 사지접합 |
| 응급내시경 | 위장관/기관지 응급내시경 |
| 중증화상 | 중증화상 전문치료 |
| 소아야간 | 달빛어린이병원 성격의 야간·휴일 소아진료 |

이 값은 병원 수용을 확정하지 않습니다. 응급 상황에서는 119, 1339 또는 병원 확인이 우선입니다.

## 달빛어린이병원 해석

달빛어린이병원은 응급실 병상 규모나 특수병상 보유량으로 평가하는 대상이 아닙니다. 야간·휴일에 소아 경증·중등도 환자가 곧바로 응급실로 몰리는 부담을 낮추고, 응급실이 중증 환자 대응에 집중할 수 있도록 돕는 보완 자원입니다.

따라서 이 프로젝트에서 일반 병원/일반 소아과는 소아 응급 접근성 판단 기준에 포함하지 않습니다.

## 정책 분석

정책 화면은 행정동 중심점과 응급기관 사이의 도로 이동시간을 사용합니다. 직선거리만으로 취약지를 판단하면 실제 도로망, 강/철도/간선도로 우회, 생활권 단절을 충분히 반영하기 어렵기 때문입니다.

| 분석 항목 | 설명 |
| --- | --- |
| 도로 이동시간 | 행정동 중심점에서 응급기관까지의 차량 이동시간 |
| 응급 접근 위험 점수 | 보호 필요 인구와 응급기관 접근시간을 함께 보는 참고 점수 |
| 평균 이동 부담 | 후보지를 추가했을 때 전체 이동시간 부담이 얼마나 줄어드는지 비교 |
| 제한 시간 안 도착 비율 | 15분/30분 안에 도착 가능한 보호 필요 인구 비율 |
| 후보 조합 | 새 거점 1~3곳을 둘 때 어떤 조합이 나은지 비교 |

현재 결과는 정책 결정을 대신하는 답이 아니라, 현장 조사와 전문가 검토 전에 우선 검토할 후보를 좁히는 참고 자료입니다.

## 실행 방법

### Backend

```bash
cd backend
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

필요한 환경 변수 예시는 다음과 같습니다.

```env
VITE_KAKAO_MAP_API_KEY=your_kakao_map_key
VITE_API_BASE_URL=http://localhost:8000
DATA_GO_KR_API_KEY=your_data_go_kr_key
KAKAO_REST_API_KEY=your_kakao_rest_key
```

### Policy Pipeline

```bash
python ai-model/run_integrated_policy_pipeline.py
```

네트워크 없이 저장된 길찾기 결과만 재사용하려면 다음 명령을 사용합니다.

```bash
python ai-model/run_integrated_policy_pipeline.py --offline
```

## 한계

- 의료진, 장비, 병상, 수용 가능 여부는 조회 시점 이후 바뀔 수 있습니다.
- 공개 길찾기 결과는 일반 차량 경로 기준이며 119 관제와 응급차 특수 운행을 반영하지 않습니다.
- 중증질환 수용 가능 정보는 공식 응답을 보여주는 참고값이며 진단 또는 이송 판단을 대체하지 않습니다.
- 병원명 매칭, 외부 서비스 장애, 정보 미제공에 따라 일부 병원은 “확인 필요”로 표시될 수 있습니다.
