# 🚑 대구 골든타임 응급의료 거버넌스 (Daegu Golden Time Governance)

> **대구광역시 행정동별 응급의료 접근성분석과 취약 인구 데이터를 융합하여 응급실 취약지를 도출하고, 실시간 병상 및 중증 응급 질환 수용성 정보를 제공하는 데이터 기반 거버넌스 플랫폼입니다.**

본 프로젝트는 단순한 병원 위치 안내를 넘어, 실제 응급상황에서 환자가 겪는 **이송 지연(응급실 뺑뺑이)** 문제와 정책결정자가 마주하는 **의료 자원 분배 최적화** 문제를 해결하기 위해 고안되었습니다.

---

## 🌟 핵심 가치 (Core Values)

*   **실시간 의료 수용성 정보 제공:** 단순 위치가 아닌 실시간 가용 병상, 의료진 부재 상황, 상황별 수용 가능(심근경색, 뇌출혈, 중증외상 등) 여부를 결합하여 제공합니다.
*   **실제 도로 네트워크 기반 분석:** 직선거리가 아닌 실제 도로망 이동시간과 소요시간을 분석하여 정밀한 공간 접근성을 도출합니다.
*   **데이터 기반 정책 의사결정 지원:** 인구 통계 데이터(소아/어르신 취약 인구)와 접근성 모델을 연계하여 최적의 신설 병원/의료 거점 후보지와 필요한 인프라(전문의, MRI 등) 보강안을 시뮬레이션합니다.

│  - Policy View : 취약 행정동 분석, 최적 입지 분석, 인프라 보강 시뮬레이터│

## 🔑 주요 기능 (Key Features)

### 1. 시민용 응급 의료 탐색 서비스 (Citizen View)
*   **실시간 병상 모니터링:** 국립중앙의료원(NEMC) 및 건강보험심사평가원(HIRA)의 API를 연계하여 실시간 가용 병상 수 및 장비 현황을 추적합니다.
*   **중증질환 수용 필터:** 심근경색(재관류중재술), 뇌출혈(수술), 중증외상, 사지절단 등 분초를 다투는 중증 질환별로 현재 수용 가능한 응급 의료기관만 선별해 보여줍니다.
*   **달빛어린이병원 분리 안내:** 소아 경증 환자가 응급실로 몰려 대기 시간이 늘어나는 현상을 막기 위해 야간·휴일 소아진료 자원인 달빛어린이병원을 별도로 필터링하여 제공합니다.

### 2. 정책 의사결정 대시보드 (Policy View)
*   **행정동별 접근 위험 분석:** 행정동별 중심점으로부터 최근접 응급실까지의 도로 이동시간과 보호 필요 취약 인구수를 종합하여 응급 접근 위험 점수(Vulnerability Score)를 지도 상에 시각화(Choropleth Map)합니다.
*   **최적 입지 후보 시뮬레이션:** K-Means 군집 모델을 기반으로 하되, 군위군 등 외곽/도시권의 공간적 편향을 보정한 **안정적 우선 검토 후보지(Stable Policy Candidates)**를 제시합니다.
*   **AI 인프라 확충 추천:** 후보지 주변(반경 5km)의 의료 인프라를 조회하여, 해당 지역에 필수적으로 확충해야 할 전문의 과목 및 특수 의료장비(MRI, CT 등)를 시뮬레이션 보고서 형태로 제공합니다.

## 🚀 빠른 시작 가이드 (Quick Start)

### 1. 환경 변수 설정
프로젝트 루트 디렉토리에 `.env` 파일을 생성하고 아래 API 키를 설정합니다.
```env
VITE_KAKAO_MAP_API_KEY=your_kakao_map_api_key
VITE_API_BASE_URL=http://localhost:8000
DATA_GO_KR_API_KEY=your_public_data_portal_key
KAKAO_REST_API_KEY=your_kakao_rest_api_key
```

### 2. 프론트엔드 실행 (Frontend)
```bash
cd frontend
npm install
npm run dev
```

### 3. 백엔드 실행 (Backend - 선택 사항)
```bash
cd backend
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

### 4. AI 정책 파이프라인 시뮬레이터 실행 (Data Pipeline)
```bash
# 전체 데이터 가공 및 정책 시뮬레이션 파이프라인 실행
python ai-model/run_integrated_policy_pipeline.py

# API 호출 쿼터를 아끼기 위해 저장된 로컬 캐시로 실행할 경우
python ai-model/run_integrated_policy_pipeline.py --offline
```

## ⚠️ 한계점 및 유의사항 (Disclaimer)
1. **실시간 정보의 시차:** 본 플랫폼에서 제공하는 의료 인력, 장비, 수용 가능 정보는 실시간 연동되나 조회 시점에 따라 실제 병원 상황과 차이가 발생할 수 있습니다. **응급 상황 시에는 반드시 119 구급대 또는 해당 병원의 안내를 따르십시오.**
