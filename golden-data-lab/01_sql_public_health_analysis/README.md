# 대구 공공보건 SQL 분석

## 프로젝트 목적
대구 행정동별 취약인구와 의료 접근성 데이터를 PostgreSQL로 분석하여 지역 간 의료 접근성 격차와 우선 관리 지역을 도출한다.

## 핵심 질문
1. 취약인구가 가장 많은 지역은 어디인가?
2. 최근접 병원까지 거리가 가장 긴 지역은 어디인가?
3. 취약인구와 접근성 위험이 동시에 높은 지역은 어디인가?
4. 소아와 고령층의 우선 관리 지역은 어떻게 다른가?

## 프로젝트 구조
```
01_sql_public_health_analysis/
├── README.md       # 프로젝트 설명 및 진행 상황 관리
├── data/           # 분석 대상 원천 데이터 파일 (CSV 등)
├── queries/        # 작성한 SQL 분석 쿼리 파일
├── results/        # 쿼리 실행 결과 (CSV, JSON 등)
└── docs/           # 데이터 명세서 및 분석 보고용 보조 자료
```
