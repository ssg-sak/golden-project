# Golden Data Lab

데이터 분석, 처리 및 시각화를 위한 프로젝트 저장소입니다.

## 폴더 구조

```
golden-data-lab/
├── README.md
├── sql/          # SQL 쿼리 및 데이터 모델링 스크립트
├── python/       # 데이터 처리 및 분석용 파이썬 스크립트
├── powerbi/      # Power BI 템플릿 및 대시보드 리소스
├── datasets/     # 원천 데이터 및 정제 데이터 저장소
├── notebooks/    # 데이터 탐색 및 프로토타이핑을 위한 Jupyter Notebooks
└── docs/         # 프로젝트 문서 및 참고 자료
```

## 시작하기

각 폴더에 필요한 리소스와 코드를 적절히 분할하여 배치해 주세요.
