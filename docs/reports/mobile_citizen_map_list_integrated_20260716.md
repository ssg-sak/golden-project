## 1. 문제 분석

### 피드백 요약

- 목록에만 보이던 병원들을 **한눈에 지도에** 보고 싶다.
- *(비유)* 네이버 지도 검색처럼 **마커가 한꺼번에** 뜨는 느낌 → **구현은 카카오맵**.
- **직선 거리만**으로는 실제 이동이 불편할 수 있다.
- 지도와 목록을 **같이** 보면서 고르고 싶다.
- 병원마다 따로 들어가야 지도를 볼 수 있어 불편하다.

### 갭

| 구분 | 이전 (모바일) | 목표 |
|------|---------------|------|
| 탐색 UI | 목록 전용 | 목록 + 지도 동시 |
| 지도 | 데스크톱(`lg+`)만 | 모바일 임베드 |
| 비교 | 거리·ETA 숫자 | 위치·방향 감각까지 |
| 상세 | 목록 → 상세만 | 지도에서 훑은 뒤 선택 |

핵심: “지도가 없다”가 아니라 **후보 위치를 한 화면에서 비교할 수 없다**.

---

## 2. 목표 · 비목표

**목표**

1. 필터된 병원 마커를 모바일 카카오맵에 일괄 표시
2. 목록과 지도를 같은 탐색 화면에서 연동
3. 거리·병상·ETA 유지 + 지도로 위치·경로 감각 보완
4. 상세 없이 후보 비교 가능
5. 직전 UX(물리 Back, 스크롤, 불투명 헤더) 유지

**비목표**

- 네이버 지도 SDK/API 도입
- Directions SDK·도로 폴리라인 API 전면 도입 (3단계는 **직선 미리보기**)
- 관리자 모바일 지도 개편

## 3. UX · 동작 규칙

```text
┌─────────────────────────┐
│ 필터 (진료대상·병상 등)   │
├─────────────────────────┤
│ 보기: 균형 / 지도 크게…  │  ← 프리셋
├─────────────────────────┤
│     카카오맵 + 마커       │  ← 높이: 프리셋 또는 드래그
│     (미리보기 직선 경로)  │
├───── 드래그 핸들 ────────┤
│ 미리보기 칩 (마커 탭 시)  │
├─────────────────────────┤
│ 목록 (거리·병상·ETA)     │
└─────────────────────────┘
        ↓ 목록 카드 또는 「상세 보기」
┌─────────────────────────┐
│ 병원 상세 (Back 유지)     │
└─────────────────────────┘
```

| 입력 | 결과 |
|------|------|
| 목록 카드 탭 | 상세 열림 (`selectedHospital`) |
| 지도 마커 탭 | 미리보기 (`previewHospital`) + 직선 Polyline · 상세는 열지 않음 |
| 미리보기 「상세 보기」 | 상세 열림 |
| 보기 프리셋 / 드래그 핸들 | 지도 높이 변경 → `ResizeObserver`·`relayout()` |
| 카카오 실패 | 목록만 폴백 |

## 4. 구현 결과 (단계별)

### 1단계 — 지도+목록 임베드

| 항목 | 내용 |
|------|------|
| 레이아웃 | 상단 카카오맵 + 하단 목록 |
| 마커 | 필터된 병원 일괄 표시 |
| 폴백 | 키 미설정·로딩·오류 시 목록만 |
| Back | `useMobileHospitalDetailHistory` 유지 |
| 데스크톱 | `lg` 분기 — 지도 단일 마운트 |

**현장 이슈:** `basis` 비율만으로는 맵 `height:100%` ≈ 0 → 빈 패널.  
**조치:** 명시적 `dvh`/픽셀 높이 + `absolute inset-0` + `onCreate` 시 `relayout()`.

### 2단계 — 보기 비율 프리셋

- 「균형 / 지도 크게 / 목록 크게」버튼으로 지도 높이 전환

### 3단계 — 드래그 + 경로 미리보기

| 항목 | 내용 |
|------|------|
| 드래그 | 지도·목록 사이 핸들로 높이 조절 (min 120px ~ 뷰포트 65%) |
| 프리셋 | 버튼 클릭 시 드래그 높이를 해당 프리셋으로 리셋 |
| 미리보기 | 마커 탭 → 내 위치↔병원 **직선** Polyline + 칩 (ETA 분 표시) |
| 고지 | UI에 「직선 미리보기」명시 — 도로 경로는 아님 |
| 상세 | 목록·칩에서만 진입 → Back UX 회귀 최소화 |
